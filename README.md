# KFTP - K. File Transfer Application

## Installation

### Download from Releases

1. Go to the [Releases](https://github.com/kty990/KFTP/releases) page
2. Download the latest installer
3. Run the executable
4. Agree to the license to finish the installation

### Install from Source
```bash
# Clone the repository
git clone https://github.com/kty990/KFTP.git
cd KFTP

# Install dependencies
npm i # npm install

# Make sure source-map-support is installed
npm i source-map-support

# Run the application
npm start
```
